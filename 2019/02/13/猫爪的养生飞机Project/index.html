管理

这可能是目前已上车的乘客最想了解的，为了防止倒卖，我将采取措施，目前人少，暂时不实施

我将会给每个使用者单独开端口并且每端口限制设备数为1或2

更加靠谱的方式

给一台VPS开有限端口数，并且每端口限制有限设备数，每端口限速。
ssr的后台功能还是很多的，因此，我可以看到你们任何连接者的IP及所在地，使用流量等等，所以我会对任何有嫌疑倒卖的端口进行极端残忍的 _限速_ 或者直接删除端口

Note

可能你们觉得我赚了，事实上在此项目前期，我是净亏，相当于免费让用。因为这个水还是很深的，墙要是稍微严格一点，就给你掐断了，目前更换的IDC是按月付费，又不像vultr按小时计费，所以整个过程都得提心吊胆的。

最后

感谢支持，我会尽量提供更优质的体验
